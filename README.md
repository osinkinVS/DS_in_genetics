**Bio** - ноутбук с получением из базы данных NCBI гена по выбранному заболеванию, его некоторых характеристик и вырезанная по координатам последовательность. 
Далее добавлена аминокислотная последовательность, количество транскриптов, количество аминокислот для самой длинной изоформы белка, полиморфизм гена, наличие или отсутсвие 3d структуры

**Bio_2_1** - секвенирование бактериального генома

**Bio_7** - Выявление мутаций
